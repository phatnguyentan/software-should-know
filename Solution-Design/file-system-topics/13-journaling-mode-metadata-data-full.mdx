# 13. Journaling mode (metadata/data/full)

> **Nhóm:** Giai đoạn 2 — Tính đúng đắn dữ liệu

## 1 vấn đề duy nhất

Hiểu đúng **Journaling mode (metadata/data/full)** để đưa ra quyết định thiết kế file system chính xác cho workload thực tế.

## 1 ví dụ thực tế

Trong hệ thống lưu trữ tài liệu nội bộ, team dùng chủ đề **Journaling mode (metadata/data/full)** để ra quyết định kỹ thuật khi tải tăng và dữ liệu tăng theo tháng.

## 3 lỗi phổ biến

1. Nhầm phạm vi áp dụng của **Journaling mode (metadata/data/full)** trong local FS và distributed FS.
2. Không đo trước/sau khi áp dụng **Journaling mode (metadata/data/full)**, dẫn tới tối ưu theo cảm tính.
3. Bỏ qua trade-off giữa hiệu năng, độ nhất quán và chi phí vận hành của **Journaling mode (metadata/data/full)**.
