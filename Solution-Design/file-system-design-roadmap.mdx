# File System Design Roadmap (Micro-Post)

> Mục tiêu: tách roadmap thành các topic nhỏ kiểu từng bài post để học dần, dễ áp dụng.

---

## Cách dùng roadmap này

- Mỗi topic = 1 bài post độc lập
- Mỗi bài chỉ tập trung 1 vấn đề
- Mỗi bài nên có: 1 sơ đồ, 1 ví dụ thực tế, 3 lỗi phổ biến, 1 checklist áp dụng
