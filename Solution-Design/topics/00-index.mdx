## Lộ Trình Với Mục Tiêu Cụ Thể

### Nếu bạn đang xây dựng AI Agent Harness:
1. Domain Modeling → Thiết kế Agent, Task, Workflow domain
2. Business Rule Design → State Machine cho agent lifecycle, Rule Engine
3. Application Architecture → Event-Driven để orchestrate agents
4. Integration → Message Queue để coordinate agents
5. Database Design → Store agent state, event history (Event Sourcing)

### Nếu bạn đang phỏng vấn System Design:
1. System Design (topic 07) — Load Balancer, Cache, Queue, CDN
2. Database Design (topic 03) — CQRS, Sharding, Replication
3. Scalability (topic 10)
4. API Design (topic 04)

### Nếu bạn đang refactor một legacy system:
1. Domain Modeling → Tìm hiểu bounded contexts
2. Application Architecture → Strangler Fig pattern
3. Database Design → Migration strategy
4. Integration → Event-driven decoupling
